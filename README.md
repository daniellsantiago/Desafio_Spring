## Como testar

### Rode a aplicação pelo Shell
 ``` shell
 $ ./mvnw spring-boot:run
 ```
Ela vai estar disponível em localhost:8080

### Usando Postman

Uma vez que a aplicação está rodando, você pode importar a coleção da API no Postman:
